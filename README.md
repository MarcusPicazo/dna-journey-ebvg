## 🇬🇧 English

**DNA Journey** is an interactive 3D web application designed to explore the molecular architecture of human DNA. Developed as a high-fidelity digital documentation tool, it allows users to navigate through a meticulously constructed 3D double helix, inspecting key biological components such as the major/minor grooves, nitrogenous bases, and the phosphate backbone.

### 🔗 Live Link
Explore the molecular structure here: [https://marcuspicazo.github.io/dna-journey-ebvg/](https://marcuspicazo.github.io/dna-journey-ebvg/)

### Technical Highlights
* **3D Rendering:** Implementation of **Three.js** for complex geometry generation, dynamic lighting, and particle environments.
* **Cinematic Animations:** Utilization of **GSAP** (GreenSock Animation Platform) to manage precise camera movements, smooth transitions, and UI interactions tied to scroll events.
* **Modern UI:** Styled with **Tailwind CSS** and custom CSS properties to achieve a sci-fi, scientific, and immersive interface.
* **Modular Architecture:** Clean separation of concerns (HTML, CSS, JS) adhering to front-end best practices and high-performance rendering.

## 🇪🇸 Español

**Viaje por el ADN** es una aplicación web interactiva 3D diseñada para explorar la arquitectura molecular del ADN humano. Desarrollado como una herramienta de documentación digital de alta fidelidad, permite a los usuarios navegar a través de una doble hélice 3D construida meticulosamente, inspeccionando componentes biológicos clave como los surcos mayor/menor, las bases nitrogenadas y el esqueleto de fosfato.

### 🔗 Enlace a la plataforma
Puedes explorar la estructura molecular en vivo aquí: [https://marcuspicazo.github.io/dna-journey-ebvg/](https://marcuspicazo.github.io/dna-journey-ebvg/)

### Puntos Técnicos Destacados
* **Renderizado 3D:** Implementación de **Three.js** para la generación de geometría compleja, iluminación dinámica y entornos de partículas.
* **Animaciones Cinematográficas:** Uso de **GSAP** para gestionar movimientos de cámara precisos, transiciones suaves e interacciones de la interfaz de usuario vinculadas al evento de scroll.
* **Interfaz de Usuario Moderna:** Estilizado con **Tailwind CSS** y propiedades CSS personalizadas para lograr una interfaz inmersiva de carácter científico y de ciencia ficción.
* **Arquitectura Modular:** Separación limpia de responsabilidades (HTML, CSS, JS) siguiendo las mejores prácticas de Front-End y optimización de rendimiento.
